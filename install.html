📲 앱으로 설치하는 법

  1. Safari(사파리)로 이 페이지를 열어요 (크롬 X)
  2. 아래 공유 버튼 􀈂(⬆️ 네모)을 눌러요
  3. 메뉴에서 "홈 화면에 추가"를 눌러요
  4. 오른쪽 위 "추가"를 누르면 끝! 🎉
  1. Chrome(크롬)으로 이 페이지를 열어요
  2. 위 "홈 화면에 앱으로 설치" 버튼을 누르거나
  3. 오른쪽 위 ⋮ 메뉴 → "앱 설치"를 눌러요
  4. "설치"를 누르면 홈 화면에 앱 아이콘 생성 🎉
