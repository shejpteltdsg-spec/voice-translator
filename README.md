# 🗣️ 실시간 통역기 (Voice Translator)

말하면 자동으로 상대 언어로 통역해서 소리 내어 읽어주는 무료 웹앱(PWA).

- 🌐 32개 언어 (한·영·일·중·불·독·서·이·포·러·아랍·힌디 등)
- 🎤 음성 인식 → 번역 → 음성 출력 (양방향)
- 📲 폰 홈 화면에 앱처럼 설치 가능 (오프라인 지원)
- 💯 완전 무료 (브라우저 내장 음성 API + MyMemory 무료 번역 API)

## 사용법
스피커폰으로 통화하면서 이 앱을 켜두세요. 내가 말할 땐 "🎤 내 언어", 상대가 말할 땐 "🎤 상대 언어" 버튼을 누르면 서로 통역됩니다.

> 참고: 마이크·음성 인식은 **안드로이드 크롬**에서 가장 잘 작동합니다.
